Definitions of key JavaScript terms

JavaScript (JS)

JavaScript (JS): A programming language readable by all browers used to add interactivity to websites.

Operators

Operators: Special characters in JavaScript that indicate an action to be performed

Variables

Variables: Variable means anything that can vary. JavaScript includes variables which hold the data value and it can be changed anytime.

Variable naming conventions

Variable naming conventions: Variables must begin with a letter, and be specific.

Functions

Functions: Code in JavaScript which performs a specific operation.

Methods

Methods: JavaScript methods are actions that can be performed on objects.

Arguments

Arguments: Are the specific parameters given to a function.
